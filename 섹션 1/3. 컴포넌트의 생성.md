
app/ex1/page.js
```
'use client'

export default function Ex1Page() {

    const message = "Hello World"

    return (
        <>
            <div className="text-3xl bg-blue-700 border m-2 p-2 text-white">
                {message}
            </div>
        </>
    )
}
```
+ use client는 브라우저라는 해석
+ <><> 태그는 jsx 의미
+ 리액트 코딩시작은 변수, 배열, 함수부터 만들기 



### 메모 정리 
+ 컴포넌트 파일 만들떄 처음에는 div부터 만들지 말 것 (<></> 태그부터 만들기)
+ 컴포넌트 파일 안에 항상 태그부터 만들지 말고, 함수, 배열, 변수 등 부터 만들기
+ TailWindCSS 편의성이 좋다.
