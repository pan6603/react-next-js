
n1/components/ex01/sampleComponent.js

```
export default function SampleComponent() {
    const num = 10

    return (
        <>
            <div>{ num % 2 === 0 ? 'EVEN' : 'ODD' }</div>
        </>
    )
}
```

n1/app/ex01/page.js

```
'use client'

import SampleComponent from "../../components/ex01/sampleComponent"

export default function Ex1Page() {

    const message = "Hello World"

    return (
        <>
            <div className="text-3xl bg-blue-700 border m-2 p-2 text-white">
                {message}
            </div>
            <SampleComponent />
        </>
    )
}
```


### 메모정리
+ Page 컴포넌트는 Next.js에서만 사용한다.
+ Page.js는 특정 경로를 호출하는 동작하는 라우팅 처리 전용 컴포넌트
+ 컴포넌트는 레고 장난감을 만들 때 코드 블록 조각
+ 파일명 짓을 때 앞에 글자 소문자로, 코드에 파일명 짓을 때 대문자로
