
### 프로젝트 생성하기 

명령어 
```
npx create-next-app@latest n1
```



## 메모 정리
+ Node.js 다운로드 하기 
+ 리액트 공식 문서 사이트에서 Next.js등 참고하기
+ Next.js 사이트 검색
+ 리액트 가능하면 최신버전 사용하기 (이유는 2025년 원격 보안 취약점이 발견됨)
+ VsCode 툴 사용한다.
+ VsCode > Material Icon Theme 설치
+ Tailwind CSS IntelliSense 설치
+ 개발할 때는 npm run dev, 배포 후 실행할 때는 npm run build
